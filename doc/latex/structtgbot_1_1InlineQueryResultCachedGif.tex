\doxysubsection{Detailed Description}
Represents a link to an animated G\+IF file stored on the Telegram servers. 

By default, this animated G\+IF file will be sent by the user with an optional caption.

Alternatively, you can use input\+\_\+message\+\_\+content to send a message with specified content instead of the animation.
