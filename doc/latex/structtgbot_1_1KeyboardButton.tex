\doxysubsection{Detailed Description}
This object represents one button of the reply keyboard. 

For simple text buttons String can be used instead of this object to specify text of the button.

Optional fields request\+\_\+contact, request\+\_\+location, and request\+\_\+poll are mutually exclusive. \begin{DoxyNote}{Note}
request\+\_\+contact and request\+\_\+location options will only work in Telegram versions released after 9 April, 2016. Older clients will display unsupported message. 

request\+\_\+poll option will only work in Telegram versions released after 23 January, 2020. Older clients will display unsupported message. 
\end{DoxyNote}
