\doxysubsection{Detailed Description}
Represents a link to an animated G\+IF file. 

By default, this animated G\+IF file will be sent by the user with optional caption.

Alternatively, you can use input\+\_\+message\+\_\+content to send a message with the specified content instead of the animation.
