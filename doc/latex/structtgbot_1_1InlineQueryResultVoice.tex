\doxysubsection{Detailed Description}
Represents a link to a voice recording in an .O\+GG container encoded with O\+P\+US. 

By default, this voice recording will be sent by the user.

Alternatively, you can use input\+\_\+message\+\_\+content to send a message with the specified content instead of the the voice message. \begin{DoxyNote}{Note}
This will only work in Telegram versions released after 9 April, 2016. Older clients will ignore them. 
\end{DoxyNote}
