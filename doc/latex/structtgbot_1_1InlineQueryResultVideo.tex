\doxysubsection{Detailed Description}
Represents a link to a page containing an embedded video player or a video file. 

By default, this video file will be sent by the user with an optional caption.

Alternatively, you can use input\+\_\+message\+\_\+content to send a message with the specified content instead of the video. \begin{DoxyNote}{Note}
If an \mbox{\hyperlink{structtgbot_1_1InlineQueryResultVideo}{Inline\+Query\+Result\+Video}} message contains an embedded video (e.\+g., You\+Tube), you must replace its content using input\+\_\+message\+\_\+content. 
\end{DoxyNote}
