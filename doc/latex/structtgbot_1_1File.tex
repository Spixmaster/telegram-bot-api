\doxysubsection{Detailed Description}
This object represents a file ready to be downloaded. The file can be downloaded via the link \href{https://api.telegram.org/file/bot}{\texttt{ https\+://api.\+telegram.\+org/file/bot}}$<$token$>$/$<$file\+\_\+path$>$. 

It is guaranteed that the link will be valid for at least 1 hour. When the link expires, a new one can be requested by calling get\+File. \begin{DoxyNote}{Note}
Maximum file size to download is 20 MB 
\end{DoxyNote}
