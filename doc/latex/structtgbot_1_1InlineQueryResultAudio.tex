\doxysubsection{Detailed Description}
Represents a link to an M\+P3 audio file. 

By default, this audio file will be sent by the user.

Alternatively, you can use input\+\_\+message\+\_\+content to send a message with the specified content instead of the audio. \begin{DoxyNote}{Note}
This will only work in Telegram versions released after 9 April, 2016. Older clients will ignore them. 
\end{DoxyNote}
