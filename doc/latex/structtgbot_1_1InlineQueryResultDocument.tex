\doxysubsection{Detailed Description}
Represents a link to a file. 

By default, this file will be sent by the user with an optional caption.

Alternatively, you can use input\+\_\+message\+\_\+content to send a message with the specified content instead of the file.

Currently, only .P\+DF and .Z\+IP files can be sent using this method. \begin{DoxyNote}{Note}
This will only work in Telegram versions released after 9 April, 2016. Older clients will ignore them. 
\end{DoxyNote}
