\doxysubsection{Detailed Description}
Represents a link to a video animation (H.\+264/\+M\+P\+E\+G-\/4 A\+VC video without sound). 

By default, this animated M\+P\+E\+G-\/4 file will be sent by the user with optional caption.

Alternatively, you can use input\+\_\+message\+\_\+content to send a message with the specified content instead of the animation.
