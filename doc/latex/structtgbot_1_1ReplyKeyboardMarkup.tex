{\footnotesize\ttfamily tgbot\+::\+Reply\+Keyboard\+Markup\+::selective}



Optional. Use this parameter if you want to show the keyboard to specific users only. 

Targets\+: 1) users that are @mentioned in the text of the \mbox{\hyperlink{structtgbot_1_1Message}{Message}} object; 2) if the bot\textquotesingle{}s message is a reply (has reply\+\_\+to\+\_\+message\+\_\+id), sender of the original message.

Example\+: A user requests to change the bot‘s language, bot replies to the request with a keyboard to select the new language. Other users in the group don’t see the keyboard.
