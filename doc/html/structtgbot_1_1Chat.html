◆ id

tgbot::Chat::id = -1

Unique identifier for this chat.

This number may be greater than 32 bits and some programming languages may have difficulty/silent defects in interpreting it.

But it is smaller than 52 bits, so a signed 64 bit integer or double-precision float type are safe for storing this identifier.

◆ invite_link

tgbot::Chat::invite_link

Optional. Chat invite link, for groups, supergroups and channel chats.

Each administrator in a chat generates their own invite links, so the bot must first generate the link using exportChatInviteLink. Returned only in getChat.
